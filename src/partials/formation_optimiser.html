Formation : optimiser les performances d'une application Web

De nos jours, développer une application web est très rapide avec la multitude d'outils mis à notre disposition. Les premiers problèmes peuvent arriver quand vous mettez cette même application en production. La performance est un aspect primordial de votre application, de votre site car dans le cas contraire, vous pouvez perdre des points dans les classements effectués par les moteurs de recherche.

Afin d'améliorer le web et l'expérience utilisateur, les différents moteurs de recherche prennent en compte les critères de performance dans l'élaboration du page rank (la note qui détermine votre classement). Quand on parle performance vous devez penser Sécurité et Rapidité. Dans cette formation je vous propose de nous focaliser sur ce second point et voir comment rendre votre site plus rapide quelque soit son contexte d'utilisation (mobilité, problème de réseaux...)

Le programme

Au cours de cette formation nous aborderons les points suivants. Chaque point sera illustré par des travaux pratiques pour vous aider à mieux comprendre l'intérêt de chaque étape

Mesurer les performances

  • Présentation des Chrome Dev Tools
  • Sites intéressants pour mesurer vos performances

Agir au niveau du protocole de communication HTTP2

Améliorer le temps de chargement initial (loadtime)

  • Limiter les ressources envoyées
  • Les images
  • Les feuilles de styles
  • Les fonts

Améliorer les temps pour le deuxième chargement (refresh / offline)

  • Mise en cache
  • Services workers

Améliorer le rendu et le premier affichage (first load)

  • Chargement de vos scripts
  • Chargement asynchrone
